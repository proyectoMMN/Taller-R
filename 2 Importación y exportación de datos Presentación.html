Directorio de trabajo

Cada archivo de Rstudio se encuentra vinculado automáticamente a una carpeta llamada directorio de trabajo.

Para saber qué carpeta está usando R como directorio de grabajo:

getwd()
## [1] "C:/Users/jorge/Desktop/USS TALLER INTRODUCCIÓN R"

Los archivos que se manejarán dentro de R deben estar en el directorio de trabajo.

Igualmente, todo archivo creado y exportado desde R llegará al directorio de trabajo.

Por defecto, el directorio de trabajo será la carpeta desde la cual se abrió el archivo de Rstudio.

Directorio de trabajo

Se puede fijar manualmente el directorio de trabajo con la función setwd():

setwd("C:/Users/jorge/Desktop/USS TALLER INTRODUCCIÓN R")

Nota: Se deben usar / en vez de \ para fijar el directorio de trabajo.

Importación de datos desde Excel

La librería readxl permite importar datos de una hoja Excel.

# Instalamos readxl

# install.library("readxl")

Una vez instalada una librería, es necesario activarla usando library().

library(readxl)
## Warning: package 'readxl' was built under R version 4.4.3

Se importan los datos con la función read_excel(), y se asignan al objeto datos_excel.

datos_excel <- read_excel("datos_inicio.xlsx")

# También se puede importar desde una carpeta específica

datos_excel <- read_excel("C:/Users/jorge/Desktop/USS TALLER INTRODUCCIÓN R/datos_inicio.xlsx")

La base de datos fue cargada exitosamente en R y se usará con el nombre de datos_excel.

Importación desde SPSS

Para importar datos desde SPSS se utiliza la función read_sav() de la librería haven.

# install.packages("haven")

library(haven)
## Warning: package 'haven' was built under R version 4.4.3
datos_de_SPSS <- read_sav("datos_inicio.sav")

Exportación de datos

Los siguientes códigos ilustran cómo exportar los datos desde R a otro formato:

Para exportar datos a formato xlsx (Excel) se usa la función write_xlsx() de la librería writexl:

# install.packages("writexl")

library(writexl)
## Warning: package 'writexl' was built under R version 4.4.3
write_xlsx(datos_excel, path = "datos_inicio2.xlsx")

Para exportar datos a formato sav (SPSS) se usa la función write_sav() de la librería haven:

write_sav(datos_excel, path = "datos_inicio2.sav")
